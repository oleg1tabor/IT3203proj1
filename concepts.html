Key Concepts

This page explains major ideas connected to Google Chrome and modern web browsing.

1. Domain Name System (DNS)

DNS is a system that helps browsers locate the correct server when the user types an address to a website. It translates domain names into the addresses needed for internet communication.

2. HTTP and HTTPS

HTTP is a protocol used to transfer web resources between a browser and a server. HTTPS is more secure protocol, which encrypts data during transmission.

3. Rendering Engine

A rendering engine reads HTML and CSS and converts them into the visible structure and style of a webpage. Chrome uses Blink as its rendering engine.

4. JavaScript Engine

A JavaScript engine runs the scripts that make webpages interactive. Chrome uses the V8 engine for fast execution of JavaScript code.

5. Sandboxing

Sandboxing is a security approach that limits what a webpage or browser process can access. This reduces damage from malicious content.

6. Progressive Web Apps (PWAs)

Progressive Web Apps are websites that behave more like installed software. They can support offline use, notifications, and app-like performance.

Concept Review List

  • HTTP and HTTPS transfer data over the web.
  • Rendering engines display page layout and style.
  • JavaScript engines support dynamic webpage behavior.
  • Sandboxing improves security.
  • PWAs show how browsers continue to evolve.
